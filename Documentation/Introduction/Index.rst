Introduction
============

What does it do?
================

The **mai_member** TYPO3 extension provides member and user management
functionality for TYPO3 13.4 LTS. It offers:

* **Member records** — an Extbase domain model with full TCA configuration
  for storing and managing member data in the TYPO3 backend.

* **Member application records** — a separate application domain model
  and service layer for handling membership application workflows.

* **Backend list module** — editors and administrators can manage
  members and applications through the standard TYPO3 list module.

* **Frontend plugin** — an Extbase/Fluid plugin for rendering
  application forms on the frontend.

* **Service layer** — ``ApplicationService`` encapsulates business logic
  for processing membership applications.
